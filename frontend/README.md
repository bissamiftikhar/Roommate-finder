
  # Roommate Matching Website

  This is a code bundle for Roommate Matching Website. The original project is available at https://www.figma.com/design/FBW33Aj9pwGAf7u6EO2Jtz/Roommate-Matching-Website.

  ## Running the code

  Run `npm i` to install the dependencies.

  Create `frontend/.env.local` with:

  ```env
  VITE_API_URL=http://localhost:3000/api
  ```

  Run `npm run dev` to start the development server.

  ### Branding
  - Place your logo at `frontend/public/logo.png`.
  - Used on Login, Register, and the dashboard sidebar avatar.
